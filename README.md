Sea Level Predictor

This project analyzes global sea level data and predicts future trends using Python, Pandas, and Matplotlib.
 📊 Features

* Scatter plot of sea level over time
* Line of best fit using all historical data
* Prediction of sea level rise until 2050
* Additional trend line using data from year 2000
 📁 Files

* `sea_level_predictor.py`
* `epa-sea-level.csv`
📥 Dataset

https://raw.githubusercontent.com/freeCodeCamp/boilerplate-sea-level-predictor/main/epa-sea-level.csv
 ▶️ Run

```bash
python sea_level_predictor.py
```
